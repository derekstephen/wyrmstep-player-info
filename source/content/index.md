## Quick Rules Reference

| Mechanic             | One Line                                                                                  |
| -------------------- | ----------------------------------------------------------------------------------------- |
| Active/Bench         | 5 through the Door, remainder run the Bastion. Both roles matter.                         |
| Quest-Board Claiming | Monday post → 24hr priority window for last session's Bench → open to all                 |
| Garrison Duty        | 2 tasks each per Bench Player, post in bastion-activities before session, roll with Avrae |
| Sundown Protocol     | Hard end time. Get out before it hits or lose everything you're carrying.                 |
| The Tithe            | 10% of all field gold goes to the Guild Vault. Automatic.                                 |
| Iron Ledger          | Everything you find belongs to the guild. You borrow it.                                  |
| Fractured State      | Death or Snap-Back = Fractured. Three tiers. Treat it early.                              |
| Memory Anchor        | One active Key writes the mission report within 72 hours. Rotate the role.                |
| Leveling             | Guild milestones only. Active and Bench advance together. No XP.                          |
