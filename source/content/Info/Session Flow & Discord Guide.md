# Session Flow & Discord Guide

This document covers everything that happens between sessions, how Discord is structured, and what you need to do and when. The week-to-week rhythm is where the guild actually operates.

---

## The Weekly Cycle

Sessions are hosted **bi-weekly on Thursdays**.

| Day | What Happens |
|---|---|
| **Thursday** | Session hosted |
| **Following Monday** | Silas posts the quest-board entry for next session |
| **Monday → Tuesday (24hrs)** | **Priority Window** — players benched last session claim Active slots first |
| **Tuesday onward** | All remaining Active slots open to everyone |
| **Before session Thursday** | Garrison Duties posted and rolled in bastion-activities |
| **Within 72hrs of session** | Memory Anchor posts mission report in mission-reports |
| **Following Monday** | Cycle repeats |

---

## Your Session Checklist

### If You Were on Bench Last Session

- [ ] Check quest-board within 24 hours of Monday post — you have the priority window
- [ ] Claim your Active slot if you want one (or volunteer Bench early if you can't attend)

### If You Are Going Active

- [ ] Check bastion-activities to see what Garrison Duty results are going into your session
- [ ] Check out items from the Guild Vault in bastion-activities before session start
- [ ] Know who the Extraction Caller is before you go through the Door
- [ ] Get out before Sundown

### If You Are on Bench

- [ ] Coordinate task selection in Discord — no duplicate tasks
- [ ] Post your two tasks in bastion-activities before session start
- [ ] Roll with Avrae in bastion-activities
- [ ] Assign your Garrison Die recipient (Task 11) if applicable — declare before session start

### After Every Session (Active Players)

- [ ] Designate the Memory Anchor before dispersing
- [ ] Return all non-soulbound items via the Extraction Ritual
- [ ] Memory Anchor: post mission report within 72 hours

## Using Avrae for Garrison Duty

Avrae handles your Garrison Duty skill checks. Before your first bench session, practice in **#🎲┃dice-testing** so you know the commands.

**Basic roll format:**
```
!r 1d20+[modifier]
```

**With advantage:**
```
!r 2d20kh1+[modifier]
```

**With disadvantage:**
```
!r 2d20kl1+[modifier]
```

Post your task selection and roll together in the session thread in bastion-activities. Include the task number, the skill you used, and the total result so the DM can apply outcomes.

## What Happens If You Miss Something

| Missed Action | Consequence |
|---|---|
| Didn't check quest-board in time | Lost priority window — slot goes to open pool |
| Didn't complete Garrison Duties | 50gp personal fine + Neglect Token. Consecutive miss = also lose priority window next session |
| Forgot to return an item | Silas retrieves it. First offense is flavor only. Repeat offense is 25gp fine. Don't repeat it. |
| No mission report posted in 72hrs | All active players lose attunements + 25gp fine per player at Tier 1. Late post removes attunement penalty but fine stands. |

## Session Day — What to Expect

**Before the session:**
- Silas announces the Sundown time in character at session start
- The countdown timer is visible on screen throughout the session
- A physical timer is at the table for in-person play
- The Extraction Caller is designated before the team goes through the Door

**During the session:**
- You will see Snap-Back glitches starting 30 minutes before Sundown
- When the Extraction Caller says it's time — it is time. No "just one more room."

**After the session:**
- Active players designate the Memory Anchor before dispersing
- All non-soulbound items return to the Guild Vault
- Memory Anchor posts the report within 72 hours

---

## Scheduling Conflicts

If you can't make a session, **volunteer for Bench as early as possible**. Message in campaign-general or DM the DM directly.

- Voluntarily benching does not cost you your priority window for next session
- Garrison Duties are expected regardless of whether you attend live
- You can complete your tasks asynchronously before session start
- If you're listening in but on Bench, you may not go through the Door

Repeated no-shows without communication affect the whole group's Bastion through Neglect Tokens and coverage gaps. Communicate early.
